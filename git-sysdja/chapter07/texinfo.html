7.11. Texinfo-7.1

Texinfo パッケージは info ページへの読み書き、変換を行うプログラムを提供します。

概算ビルド時間: 0.2 SBU
必要ディスク容量: 130 MB

7.11.1. Texinfo のインストール

Texinfo をコンパイルするための準備をします。 

./configure --prefix=/usr

パッケージをコンパイルします。 

make

パッケージをインストールします。 

make install

本パッケージの詳細は 「Texinfo の構成」を参照してください。
