6.14. Sed-4.9

Sed パッケージはストリームエディターを提供します。

概算ビルド時間: 0.1 SBU
必要ディスク容量: 21 MB

6.14.1. Sed のインストール

Sed をコンパイルするための準備をします。 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --build=$(./build-aux/config.guess)

パッケージをコンパイルします。 

make

パッケージをインストールします。 

make DESTDIR=$LFS install

本パッケージの詳細は 「Sed の構成」を参照してください。
