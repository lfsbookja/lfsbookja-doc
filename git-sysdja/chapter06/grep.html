6.10. Grep-3.11

Grep パッケージはファイル内の検索を行うプログラムを提供します。

概算ビルド時間: 0.1 SBU
必要ディスク容量: 27 MB

6.10.1. Grep のインストール

Grep をコンパイルするための準備をします。 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --build=$(./build-aux/config.guess)

パッケージをコンパイルします。 

make

パッケージをインストールします。 

make DESTDIR=$LFS install

本パッケージの詳細は 「Grep の構成」を参照してください。
