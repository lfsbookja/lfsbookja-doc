6.11. Gzip-1.13

Gzip パッケージはファイルの圧縮、伸長 (解凍) を行うプログラムを提供します。

概算ビルド時間: 0.1 SBU
必要ディスク容量: 11 MB

6.11.1. Gzip のインストール

Gzip をコンパイルするための準備をします。 

./configure --prefix=/usr --host=$LFS_TGT

パッケージをコンパイルします。 

make

パッケージをインストールします。 

make DESTDIR=$LFS install

本パッケージの詳細は 「Gzip の構成」を参照してください。
