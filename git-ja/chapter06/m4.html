6.2. M4-1.4.19

M4 パッケージはマクロプロセッサーを提供します。

概算ビルド時間: 0.1 SBU
必要ディスク容量: 32 MB

6.2.1. M4 のインストール

M4 をコンパイルするための準備をします。 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --build=$(build-aux/config.guess)

パッケージをコンパイルします。 

make

パッケージをインストールします。 

make DESTDIR=$LFS install

本パッケージの詳細は 「M4 の構成」を参照してください。
