6.12. Make-4.4.1

Make パッケージは、対象となるパッケージのソースファイルを用いて、実行モジュールやそれ以外のファイルの生成、管理を行うプログラムを提供します。

概算ビルド時間: 0.1 SBU 以下
必要ディスク容量: 15 MB

6.12.1. Make のインストール

Make をコンパイルするための準備をします。 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --build=$(build-aux/config.guess)

パッケージをコンパイルします。 

make

パッケージをインストールします。 

make DESTDIR=$LFS install

本パッケージの詳細は 「Make の構成」を参照してください。
