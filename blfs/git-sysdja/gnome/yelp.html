Yelp-42.2

Yelp の概要

Yelp パッケージは、ヘルプファイルを参照するためのヘルプブラウザーを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.gnome.org/sources/yelp/42/yelp-42.2.tar.xz

  • ダウンロード MD5 sum: 3792122c4ab90725716cd88e9274f0f6

  • ダウンロードサイズ: 1.4 MB

  • 必要ディスク容量: 21 MB

  • 概算ビルド時間: 0.1 SBU

Yelp の依存パッケージ

必須

gsettings-desktop-schemas-45.0, WebKitGTK-2.42.4, yelp-xsl-42.1

推奨

desktop-file-utils-0.27

任意

GTK-Doc-1.33.2

注記

The Yelp package is not required for a functional GNOME Desktop. Note, however, that without Yelp you will not be able to view the built-in Help provided by core GNOME and many of the support applications.

Yelp のインストール

以下のコマンドを実行して Yelp をビルドします。 

./configure --prefix=/usr --disable-static &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

Still as the root user, ensure that /usr/share/applications/mimeinfo.cache will be updated and therefore the help in gnome applications will work. 

update-desktop-database

コマンド説明

--disable-static: このスイッチはスタティックライブラリをインストールしないようにします。

--enable-gtk-doc: GTK-Doc をインストール済で API ドキュメントを再構築する場合に指定します。

パッケージ構成

インストールプログラム: gnome-help (シンボリックリンク), yelp
インストールライブラリ: libyelp.so
インストールディレクトリ: /usr/include/libyelp, /usr/share/gtk-doc/html/libyelp (任意インストール), /usr/{lib,share}/yelp

概略説明

yelp

GNOME ヘルプブラウザー。

libyelp.so

Yelp API 関数を提供します。
