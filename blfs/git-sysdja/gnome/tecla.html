Tecla-45.0

Tecla の概要

The Tecla package contains a keyboard layout viewer.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.gnome.org/sources/tecla/45/tecla-45.0.tar.xz

  • ダウンロード MD5 sum: 68a44119c5c76ca952a4cb6ca2e0fd22

  • ダウンロードサイズ: 36 KB

  • 必要ディスク容量: 2.0 MB

  • 概算ビルド時間: 0.1 SBU 以下

Tecla の依存パッケージ

必須

libadwaita-1.4.2, libxkbcommon-1.6.0

Tecla のインストール

以下のコマンドを実行して Tecla をビルドします。 

mkdir build &&
cd    build &&

meson setup --prefix=/usr --buildtype=release .. &&
ninja

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

ninja install

コマンド説明

--buildtype=release: 本パッケージの安定版リリース向けに適したビルドタイプを指定します。 デフォルトにしていると、最適化されていないバイナリを生成する場合があるためです。

パッケージ構成

インストールプログラム: tecla
インストールライブラリ: なし
インストールディレクトリ: なし
