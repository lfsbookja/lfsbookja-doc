gnome-maps-45.2

GNOME Maps の概要

GNOME Maps は GNOME におけるマップアプリケーション（map application）です。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.gnome.org/sources/gnome-maps/45/gnome-maps-45.2.tar.xz

  • ダウンロード MD5 sum: bf3276eabeaf261a0d8ccfdd4584d245

  • ダウンロードサイズ: 3.0 MB

  • 必要ディスク容量: 16 MB

  • 概算ビルド時間: 0.1 SBU

GNOME Maps の依存パッケージ

必須

desktop-file-utils-0.27, GeoClue-2.7.1, geocode-glib-3.26.4, Gjs-1.78.1, libadwaita-1.4.2, libshumate-1.1.2, libgweather-4.4.0, rest-0.9.1

GNOME Maps のインストール

以下のコマンドを実行して GNOME Maps をビルドします。 

mkdir build &&
cd    build &&

meson setup --prefix=/usr --buildtype=release .. &&
ninja

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

ninja install

パッケージ構成

インストールプログラム: gnome-maps (シンボリックリンク)
インストールライブラリ: libgnome-maps.so
インストールディレクトリ: /usr/lib/gnome-maps, /usr/share/gnome-maps
