gnome-logs-43.0

GNOME Logs の概要

The GNOME Logs package contains a log viewer for the systemd journal.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.gnome.org/sources/gnome-logs/43/gnome-logs-43.0.tar.xz

  • ダウンロード MD5 sum: fbeada138a28d7ba13a4a95d585ee4c0

  • ダウンロードサイズ: 576 KB

  • 必要ディスク容量: 8 MB

  • 概算ビルド時間: 0.1 SBU

GNOME Logs の依存パッケージ

必須

GTK-4.12.4, gsettings-desktop-schemas-45.0, itstool-2.0.7, libadwaita-1.4.2

任意

desktop-file-utils-0.27 (for the test suite), docbook-xml-4.5, docbook-xsl-nons-1.79.2, libxslt-1.1.39 (to build manual pages), appstream-glib, dogtail

GNOME Logs のインストール

以下のコマンドを実行して GNOME Logs をビルドします。 

mkdir build &&
cd    build &&

meson setup --prefix=/usr --buildtype=release .. &&
ninja

ビルド結果をテストする場合は ninja test を実行します。

root ユーザーになって以下を実行します。 

ninja install

コマンド説明

--buildtype=release: 本パッケージの安定版リリース向けに適したビルドタイプを指定します。 デフォルトにしていると、最適化されていないバイナリを生成する場合があるためです。

パッケージ構成

インストールプログラム: gnome-logs
インストールライブラリ: なし
インストールディレクトリ: /usr/share/gnome-logs, /usr/share/help/*/gnome-logs

概略説明
