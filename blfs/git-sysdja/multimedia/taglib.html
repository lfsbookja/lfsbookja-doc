Taglib-1.13.1

Taglib の概要

Taglib は、オーディオファイルのタグの入出力および操作を行うライブラリです。 Amarok や VLC などのアプリケーションがこれを利用しています。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://taglib.org/releases/taglib-1.13.1.tar.gz

  • ダウンロード MD5 sum: 2fe6089da73ad414aa1b982b83415362

  • ダウンロードサイズ: 1.3 MB

  • 必要ディスク容量: 13 MB

  • 概算ビルド時間: 0.2 SBU

Taglib の依存パッケージ

必須

CMake-3.28.1

任意 (テストのため)

Cppunit

Taglib のインストール

以下のコマンドを実行して Taglib をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      -DBUILD_SHARED_LIBS=ON \
      .. &&
make

The test suite requires Cppunit but it's not a BLFS package.

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: taglib-config
インストールライブラリ: libtag.so and libtag_c.so
インストールディレクトリ: /usr/include/taglib

Short Descriptions

taglib-config

is a tool used to print information about the taglib installation
