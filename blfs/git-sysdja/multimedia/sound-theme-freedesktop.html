sound-theme-freedesktop-0.8

Sound Theme Freedesktop の概要

Sound Theme Freedesktop パッケージは、デスクトップにおける音楽テーマ (sound themes) を提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://people.freedesktop.org/~mccann/dist/sound-theme-freedesktop-0.8.tar.bz2

  • ダウンロード MD5 sum: d7387912cfd275282d1ec94483cb2f62

  • ダウンロードサイズ: 472 KB

  • 必要ディスク容量: 2 MB

  • 概算ビルド時間: 0.1 SBU 以下

Sound Theme Freedesktop のインストール

以下のコマンドを実行して Sound Theme Freedesktop をビルドします。 

./configure --prefix=/usr &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: なし
インストールディレクトリ: /usr/share/sounds/freedesktop
