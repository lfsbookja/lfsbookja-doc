libsamplerate-0.2.2

libsamplerate の概要

libsamplerate は音声データに対するサンプルレートコンバーターです。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://github.com/libsndfile/libsamplerate/releases/download/0.2.2/libsamplerate-0.2.2.tar.xz

  • ダウンロード MD5 sum: 97c010fc25156c33cddc272c1935afab

  • ダウンロードサイズ: 3.2 MB

  • 必要ディスク容量: 15 MB (add 2 MB for tests)

  • 概算ビルド時間: less than 0.1 SBU (add 0.3 SBU for tests)

libsamplerate の依存パッケージ

任意

alsa-lib-1.2.10, libsndfile-1.2.2, fftw-3.3.10 (テスト実行時)

libsamplerate のインストール

以下のコマンドを実行して libsamplerate をビルドします。 

./configure --prefix=/usr    \
            --disable-static \
            --docdir=/usr/share/doc/libsamplerate-0.2.2 &&
make

ビルド結果をテストする場合は make check を実行します。

root ユーザーになって以下を実行します。 

make install

コマンド説明

--disable-static: このスイッチはスタティックライブラリをインストールしないようにします。

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libsamplerate.so
インストールディレクトリ: /usr/share/doc/libsamplerate-0.2.2
