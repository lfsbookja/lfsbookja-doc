Parole-4.18.1

Parole の概要

Parole は Xfce 上において GStreamer を用いた DVD/CD 音楽プレイヤーです。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

注記

Although this version of parole works with most files, it is unable to play DVDs, reporting an error in the gstreamer backend.

パッケージ情報

  • ダウンロード (HTTP): https://archive.xfce.org/src/apps/parole/4.18/parole-4.18.1.tar.bz2

  • ダウンロード MD5 sum: 29a409b4b22c2d91f210679e5708a19d

  • ダウンロードサイズ: 916 KB

  • 必要ディスク容量: 19 MB

  • 概算ビルド時間: 0.2 SBU

Parole の依存パッケージ

必須

dbus-glib-0.112, gst-plugins-base-1.22.7, gst-plugins-good-1.22.7, libxfce4ui-4.18.4

推奨

libnotify-0.8.3, taglib-1.13.1

Optional

GTK-Doc-1.33.2

Parole のインストール

以下のコマンドを実行して Parole をビルドします。 

./configure --prefix=/usr &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

Parole の利用

Gstreamer Plugins Ugly をインストールしていて、libdvdnav と libdvdread のサポートが有効である場合、 Parole による DVD 再生は Media>Open location を実行し、入力欄に dvd:// を入力します。

同じく CD 再生は Media>Open location を実行し、入力欄に cdda:// を入力します。

パッケージ構成

インストールプログラム: parole
インストールライブラリ: /usr/lib/parole-0/ 配下に二つのライブラリ
インストールディレクトリ: /usr/include/parole, /usr/lib/parole-0, /usr/share/parole

概略説明

parole

GStreamer を利用した GTK+ 3 メディアプレイヤー。
