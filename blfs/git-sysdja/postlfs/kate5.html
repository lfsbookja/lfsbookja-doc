Kate-23.08.4

Kate の概要

The Kate package contains an advanced KF5 based graphical text editor.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/release-service/23.08.4/src/kate-23.08.4.tar.xz

  • ダウンロード MD5 sum: cd727192a096531f798746f4021289e1

  • ダウンロードサイズ: 7.7 MB

  • 必要ディスク容量: 211 MB

  • 概算ビルド時間: 0.8 SBU (using parallelism=4)

Kate の依存パッケージ

必須

KDE Frameworks-5.113.0

Optional

libgit2

Kate のインストール

以下のコマンドを実行して Kate をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX  \
      -DCMAKE_BUILD_TYPE=Release          \
      -DBUILD_TESTING=OFF                 \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: kate, kwrite
インストールライブラリ: Several plugins under $KF5_PREFIX/lib/plugins
インストールディレクトリ: $KF5_PREFIX/lib/plugins/ktexteditor, $KF5_PREFIX/lib/plugins/plasma/dataengine, $KF5_PREFIX/share/doc/HTML/*/{kate,katepart,kwrite}, $KF5_PREFIX/share/{kateproject,katexmltools}, $KF5_PREFIX/share/plasma/plasmoids/org.kde.plasma.katesessions

概略説明

kate

is an advanced text editor for kde

kwrite

is a text editor for KDE, that is a light version of kate
