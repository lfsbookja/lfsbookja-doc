xbitmaps-1.1.3

xbitmaps の概要

xbitmaps は、Xorg の章においてビルドする複数のアプリケーションが、ビットマップを描画するために必要となるスタティックなグラフィックを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://www.x.org/pub/individual/data/xbitmaps-1.1.3.tar.xz

  • ダウンロード MD5 sum: 2b03f89d78fb91671370e77d7ad46907

  • ダウンロードサイズ: 108 KB

  • 必要ディスク容量: 1.6 MB

  • 概算ビルド時間: 0.1 SBU 以下

xbitmaps の依存パッケージ

必須

util-macros-1.20.0

xbitmaps のインストール

以下のコマンドを実行して xbitmaps パッケージをビルドします。 

./configure $XORG_CONFIG

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: なし
インストールディレクトリ: $XORG_PREFIX/include/X11/bitmaps
