hicolor-icon-theme-0.17

hicolor-icon-theme の概要

hicolor-icon-theme パッケージは、アイコンテーマ仕様に基づいて実現されたソフトウェアに対して、デフォルトテーマ (fallback theme) を提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://icon-theme.freedesktop.org/releases/hicolor-icon-theme-0.17.tar.xz

  • ダウンロード MD5 sum: 84eec8d6f810240a069c731f1870b474

  • ダウンロードサイズ: 52 KB

  • 必要ディスク容量: 340 KB

  • 概算ビルド時間: 0.1 SBU 以下

hicolor-icon-theme のインストール

以下のコマンドを実行して hicolor-icon-theme をビルドします。 

./configure --prefix=/usr

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: なし
インストールディレクトリ: /usr/share/icons/hicolor

概略説明

/usr/share/icons/hicolor/*

デフォルトで用いられるアイコン定義を提供します。
