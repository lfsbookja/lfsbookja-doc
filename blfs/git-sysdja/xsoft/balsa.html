Balsa-2.6.4

Balsa の概要

Balsa パッケージは GNOME-2 ベースのメールクライアントを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://pawsa.fedorapeople.org/balsa/balsa-2.6.4.tar.xz

  • ダウンロード MD5 sum: bebd22cb6bac87ee05255c48ee1d28dd

  • ダウンロードサイズ: 3.5 MB

  • 必要ディスク容量: 66 MB

  • 概算ビルド時間: 0.2 SBU (Using parallelism=4)

Additional Downloads

  • Required patch: https://www.linuxfromscratch.org/patches/blfs/svn/balsa-2.6.4-upstream_fixes-2.patch

Balsa の依存パッケージ

必須

Aspell-0.60.8.1, enchant-2.6.4, GMime-3.2.7, GPGME-1.23.2, GTK+-3.24.38, libnotify-0.8.3

必須 (実行時)

MTA (that provides a sendmail command)

任意

Compface-1.5.2, Gcr-3.41.1, gtksourceview-3.24.11, libcanberra-0.30, libsecret-0.21.2, MIT Kerberos V5-1.21.2, OpenLDAP-2.6.6, Procmail-3.22, SQLite-3.44.2, WebKitGTK-2.42.4, html2text, osmo, rubrica, yelp-tools

Balsa のインストール

First, fix build issues when configuring without WebKitGTK, or when using WebKitGTK-2.38 or later. 

patch -Np1 -i ../balsa-2.6.4-upstream_fixes-2.patch

以下のコマンドを実行して Balsa をビルドします。 

./configure --prefix=/usr            \
            --sysconfdir=/etc        \
            --localstatedir=/var/lib \
            --without-html-widget    &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

コマンド説明

--without-html-widget: This option disables building the HTML renderer. If you want the HTML renderer to be built, verify that you have WebKitGTK-2.42.4 installed, and then remove this option.

--with-canberra: Enable libcanberra support.

--with-compface: Enable compface support.

--with-gcr: Use libgcr for TLS support.

--with-gss: Enable GSSAPI support.

--with-ldap: Use this option to enable LDAP address book support if OpenLDAP is installed.

--with-libsecret: Enable strong password support and support for gnome-keyring to store passwords.

--with-sqlite: Use this option to enable SQLite address book support if SQLite is installed.

パッケージ構成

インストールプログラム: balsa and balsa-ab
インストールライブラリ: なし
インストールディレクトリ: /etc/sounds/events, /usr/share/balsa, and /usr/share/sounds/balsa

概略説明

balsa

glib ベースのメールクライアント。

balsa-ab

is the address-book application used by balsa
