lxqt-notificationd-1.4.0

lxqt-notificationd の概要

The lxqt-notificationd package is the LXQt notification daemon.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://github.com/lxqt/lxqt-notificationd/releases/download/1.4.0/lxqt-notificationd-1.4.0.tar.xz

  • ダウンロード MD5 sum: 823aef2b5ad40b332a3ea6f83488bdad

  • ダウンロードサイズ: 64 KB

  • 必要ディスク容量: 4.3 MB

  • 概算ビルド時間: 0.2 SBU

lxqt-notificationd の依存パッケージ

必須

liblxqt-1.4.0

lxqt-notificationd のインストール

以下のコマンドを実行して lxqt-notificationd をビルドします。 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_BUILD_TYPE=Release  \
      -DCMAKE_INSTALL_PREFIX=/usr \
      ..       &&

make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: lxqt-config-notificationd, lxqt-notificationd
インストールライブラリ: なし
インストールディレクトリ: /usr/share/lxqt/translations/lxqt-config-notificationd, /usr/share/lxqt/translations/lxqt-notificationd

概略説明

lxqt-config-notificationd

is the lxqt-notificationd GUI configuration tool

lxqt-notificationd

is the LXQt notification daemon
