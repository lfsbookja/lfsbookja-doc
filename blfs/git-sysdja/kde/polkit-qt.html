Polkit-Qt-0.114.0

Polkit-Qt の概要

Polkit-Qt は Qt 環境での PolicyKit に対する API を提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/polkit-qt-1/polkit-qt-1-0.114.0.tar.xz

  • ダウンロード MD5 sum: 2d20b02bd7ad1cd3f6ce235fc6d7de60

  • ダウンロードサイズ: 60 KB

  • 必要ディスク容量: 3.5 MB

  • 概算ビルド時間: 0.1 SBU

Polkit-Qt の依存パッケージ

必須

CMake-3.28.1, Polkit-123, Qt-5.15.11

Polkit-Qt のインストール

以下のコマンドを実行して Polkit-Qt をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

コマンド説明

-DCMAKE_BUILD_TYPE=Release: このスイッチは、コンパイラーの最適化オプションを最高レベルにて適用することを指示します。

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libpolkit-qt5-agent-1.so, libpolkit-qt5-core-1.so, libpolkit-qt5-gui-1.so
インストールディレクトリ: /usr/include/polkit-qt5-1, /usr/lib/cmake/PolkitQt5-1
