libkexiv2-23.08.4

libkexiv2 の概要

Libkexiv2 は、イメージメタデータを操作するための Exiv2 をラップする KDE ライブラリです。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/release-service/23.08.4/src/libkexiv2-23.08.4.tar.xz

  • ダウンロード MD5 sum: 9572c8f2401187ca63ab6a40121ecf48

  • ダウンロードサイズ: 60 KB

  • 必要ディスク容量: 3.3 MB

  • 概算ビルド時間: 0.2 SBU

libkexiv2 の依存パッケージ

必須

KDE Frameworks-5.113.0, Exiv2-0.28.1

libkexiv2 のインストール

以下のコマンドを実行して libkexiv2 をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーとなって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libKF5KExiv2.so
インストールディレクトリ: $KF5_PREFIX/include/KF5/KExiv2, $KF5_PREFIX/lib/cmake/KF5KExiv2
