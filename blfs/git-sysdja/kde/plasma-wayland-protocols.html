Plasma-wayland-protocols-1.12.0

Plasma-wayland-protocols の概要

本パッケージは KDE のプロトコル定義のカスタムセットを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/plasma-wayland-protocols/plasma-wayland-protocols-1.12.0.tar.xz

  • ダウンロード MD5 sum: 7755dcac14e90068a7e26922c3a4a296

  • ダウンロードサイズ: 44 KB

  • 必要ディスク容量: 740 KB

  • 概算ビルド時間: 0.1 SBU 以下

Plasma-wayland-protocols の依存パッケージ

必須

extra-cmake-modules-5.113.0

Plasma-wayland-protocols のインストール

以下のコマンドを実行して Plasma-wayland-protocols をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr ..

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: なし
インストールディレクトリ: /usr/share/plasma-wayland-protocols, /usr/lib/cmake/PlasmaWaylandProtocols
