Wayland-Protocols-1.32

Wayland-Protocols の概要

The Wayland-Protocols package contains additional Wayland protocols that add functionality outside of protocols already in the Wayland core.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://gitlab.freedesktop.org/wayland/wayland-protocols/-/releases/1.32/downloads/wayland-protocols-1.32.tar.xz

  • ダウンロード MD5 sum: 00c2cedb0d2df714a0965a00c19385c6

  • ダウンロードサイズ: 92 KB

  • 必要ディスク容量: 7.4 MB (テスト込み)

  • 概算ビルド時間: 0.1 SBU 以下 (テスト込み)

Wayland-protocols の依存パッケージ

必須

Wayland-1.22.0

Wayland-protocols のインストール

以下のコマンドを実行して Wayland-protocols をビルドします。 

mkdir build &&
cd    build &&

meson setup --prefix=/usr --buildtype=release &&
ninja

ビルド結果をテストする場合は ninja test を実行します。

root ユーザーになって以下を実行します。 

ninja install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: なし
インストールディレクトリ: /usr/share/wayland-protocols
