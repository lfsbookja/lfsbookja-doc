libdaemon-0.14

libdaemon の概要

libdaemon パッケージは、UNIX デーモンを構築するための軽量な C ライブラリを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://0pointer.de/lennart/projects/libdaemon/libdaemon-0.14.tar.gz

  • ダウンロード MD5 sum: 509dc27107c21bcd9fbf2f95f5669563

  • ダウンロードサイズ: 332 KB

  • 必要ディスク容量: 3 MB

  • 概算ビルド時間: 0.1 SBU

libdaemon の依存パッケージ

任意

Doxygen-1.9.8, Lynx-2.8.9rel.1

libdaemon のインストール

以下のコマンドを実行して libdaemon をビルドします。 

./configure --prefix=/usr --disable-static &&
make

Doxygen をインストールしていて API ドキュメントをする場合は、以下のコマンドを実行します。 

make -C doc doxygen

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make docdir=/usr/share/doc/libdaemon-0.14 install

API ドキュメントをビルドした場合は root ユーザーになり、以下のコマンドを実行してドキュメントをインストールします。 

install -v -m755 -d /usr/share/doc/libdaemon-0.14/reference/html &&
install -v -m644 doc/reference/html/* /usr/share/doc/libdaemon-0.14/reference/html &&
install -v -m644 doc/reference/man/man3/* /usr/share/man/man3

コマンド説明

--disable-static: このスイッチはスタティックライブラリをインストールしないようにします。

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libdaemon.so
インストールディレクトリ: /usr/include/libdaemon, /usr/share/doc/libdaemon-0.14
