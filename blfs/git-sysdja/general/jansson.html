Jansson-2.14

Jansson の概要

Jansson パッケージは JSON データのエンコード、デコードなどの操作を行うライブラリを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://github.com/akheron/jansson/releases/download/v2.14/jansson-2.14.tar.bz2

  • ダウンロード MD5 sum: 3f90473d7d54ebd1cb6a2757396641df

  • ダウンロードサイズ: 424 KB

  • 必要ディスク容量: 5.6 MB (add 1.9 MB for tests)

  • 概算ビルド時間: 0.1 SBU (with tests)

Jansson のインストール

First fix one of the tests: 

sed -e "/DT/s;| sort;| sed 's/@@libjansson.*//' &;" \
    -i test/suites/api/check-exports

以下のコマンドを実行して jansson をビルドします。 

./configure --prefix=/usr --disable-static &&
make

ビルド結果をテストする場合は make check を実行します。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libjansson.so
インストールディレクトリ: なし

概略説明

libjansson.so

JSON データのエンコード、デコードなどを行う API を提供します。
