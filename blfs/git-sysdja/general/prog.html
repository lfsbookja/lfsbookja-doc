プログラミング言語

ベースシステムとしている LFS でも開発環境として利用することはできます。 しかし LFS には C, C++, Perl, Python しか含まれていません。 本章では代表的なプログラミング環境を構築する手順を説明し、システム開発力を大いに繰り広げていただくことを目指しています。
