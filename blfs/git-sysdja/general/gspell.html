gspell-1.12.2

gspell の概要

gspell パッケージは、GTK+ アプリケーションに対してスペルチェック機能を追加するための柔軟な API を提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.gnome.org/sources/gspell/1.12/gspell-1.12.2.tar.xz

  • ダウンロード MD5 sum: f1e5f02695aee20ba543352889c28ff5

  • ダウンロードサイズ: 436 KB

  • 必要ディスク容量: 11 MB (テスト込み)

  • 概算ビルド時間: 0.1 SBU (テスト込み)

gspell の依存パッケージ

必須

enchant-2.6.4, ICU-74.1, GTK+-3.24.38

任意

gobject-introspection-1.78.1, GTK-Doc-1.33.2, Vala-0.56.14, Valgrind-3.22.0

gspell のインストール

以下のコマンドを実行して gspell をビルドします。 

./configure --prefix=/usr &&
make

ビルド結果をテストする場合は make check を実行します。 テストは X セッション内において実行することが必要です。 test-checker というテストが、外部パッケージである Hunspell がインストールされていない場合には失敗します。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: gspell-app1
インストールライブラリ: libgspell-1.so
インストールディレクトリ: /usr/include/gspell-1, /usr/share/gtk-doc/html/gspell-1.0

Short Descriptions

gspell-app1

checks the spelling of a text entered in a window

libgspell-1.so

is the gspell API library
