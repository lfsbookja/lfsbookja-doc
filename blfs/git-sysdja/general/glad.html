Glad-2.0.4

Glad の概要

The Glad package contains a generator for loading Vulkan, OpenGL, EGL, GLES, and GLX contexts.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://github.com/Dav1dde/glad/archive/refs/tags/v2.0.4/glad-2.0.4.tar.gz

  • ダウンロード MD5 sum: 154151c79dab047b26db52fe0572a409

  • ダウンロードサイズ: 588 KB

  • 必要ディスク容量: 36 MB

  • 概算ビルド時間: 0.1 SBU 以下

Glad の依存パッケージ

任意 (テスト実行のために必要)

pytest-7.4.3, rustc-1.74.1, Xorg ライブラリ, glfw, WINE

Glad のインストール

以下のコマンドを実行して Glad をビルドします。 

pip3 wheel -w dist --no-build-isolation --no-deps --no-cache-dir $PWD

本パッケージにはテストスイートがあります。 ただしこれを実行するためには、上に示した外部依存パッケージのインストールが必要です。

root ユーザーになって以下を実行します。 

pip3 install --no-index --find-links=dist --no-cache-dir --no-user glad2

パッケージ構成

インストールプログラム: glad
インストールライブラリ: なし
インストールディレクトリ: /usr/lib/python3.12/site-packages/glad, /usr/lib/python3.12/site-packages/glad2-2.0.4.dist-info
