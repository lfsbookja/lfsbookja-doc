libnsl-2.0.1

libnsl の概要

libnsl パッケージは、NIS（YP）に対するクライアントインターフェースを提供します。 これは glibc にて利用されている NIS ライブラリに置き換わるものです。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://github.com/thkukuk/libnsl/releases/download/v2.0.1/libnsl-2.0.1.tar.xz

  • ダウンロード MD5 sum: fb178645dfa85ebab0f1e42e219b42ae

  • ダウンロードサイズ: 276 KB

  • 必要ディスク容量: 3.1 MB

  • 概算ビルド時間: less than 0.1 SBU

libnsl の依存パッケージ

必須

libtirpc-1.3.4

libnsl のインストール

以下のコマンドを実行して libnsl をビルドします。 

./configure --sysconfdir=/etc --disable-static &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

コマンド説明

--disable-static: このスイッチはスタティックライブラリをインストールしないようにします。

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libnsl.so
インストールディレクトリ: /usr/include/rpcsvc

概略説明

libnsl.so

他のプログラムが必要とする NIS (YP) API 関数を提供します。
