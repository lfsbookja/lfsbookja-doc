Links-2.29

Links の概要

Links は、テキストモードおよびグラフィックモードを持つ WWW ブラウザーです。 テーブルやフレームのレンダリング、バックグラウンドダウンロード、カラー表示など、数多くの機能をサポートしています。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): http://links.twibright.com/download/links-2.29.tar.bz2

  • ダウンロード MD5 sum: f60b87ffee090c5d6820951eba710572

  • ダウンロードサイズ: 6.2 MB

  • 必要ディスク容量: 35 MB

  • 概算ビルド時間: 0.2 SBU

Links の依存パッケージ

推奨

libevent-2.1.12

任意

Graphics mode requires at least one of GPM-1.20.7 (mouse support to be used with a framebuffer-based console), SVGAlib, DirectFB , and グラフィック環境

さまざまなイメージフォーマットのデコーディングを行うために Links では以下を利用します。 libavif-1.0.3, libpng-1.6.40, libjpeg-turbo-3.0.1, librsvg-2.57.1, libtiff-4.6.0

For decompressing web pages that are compressed with Brotli, Links can utilize Brotli-1.1.0

Links のインストール

以下のコマンドを実行して Links をビルドします。 

./configure --prefix=/usr --mandir=/usr/share/man &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install &&
install -v -d -m755 /usr/share/doc/links-2.29 &&
install -v -m644 doc/links_cal/* KEYS BRAILLE_HOWTO \
    /usr/share/doc/links-2.29

コマンド説明

--enable-graphics: このパラメーターはグラフィックモードのサポートを有効にします。

Links の設定

設定ファイル

~/.links/*

設定情報

Links では、各ユーザーごとの設定ファイルが ~/.links ディレクトリに保存されます。 このファイルは links が初めて起動した際に自動的に生成されます。

パッケージ構成

インストールプログラム: links
インストールライブラリ: なし
インストールディレクトリ: /usr/share/doc/links-2.29
