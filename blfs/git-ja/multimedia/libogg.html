libogg-1.3.5

libogg の概要

libogg パッケージは、Ogg ファイル構造 (Ogg file structure) を提供します。 これは単一の物理ビットストリームの生成 (エンコーディング) と再生 (デコーディング) を実現します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://downloads.xiph.org/releases/ogg/libogg-1.3.5.tar.xz

  • ダウンロード MD5 sum: 3178c98341559657a15b185bf5d700a5

  • ダウンロードサイズ: 420 KB

  • 必要ディスク容量: 3.5 MB （テスト込み）

  • 概算ビルド時間: 0.1 SBU 以下 （テスト込み）

libogg のインストール

以下のコマンドを実行して libogg をビルドします。 

./configure --prefix=/usr    \
            --disable-static \
            --docdir=/usr/share/doc/libogg-1.3.5 &&
make

ビルド結果をテストする場合は make check を実行します。

root ユーザーになって以下を実行します。 

make install

コマンド説明

--disable-static: このスイッチはスタティックライブラリをインストールしないようにします。

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libogg.so
インストールディレクトリ: /usr/include/ogg, /usr/share/doc/libogg-1.3.5

概略説明

libogg.so

Ogg が定める書式のビットストリームを取り扱うための関数を提供します。
