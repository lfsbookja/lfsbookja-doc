kwave-23.08.4

KWave の概要

KWave パッケージは KF5 ベースのサウンドエディターアプリケーションを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/release-service/23.08.4/src/kwave-23.08.4.tar.xz

  • ダウンロード MD5 sum: cf29fd3e346f9fe7c8caadd89efa6b56

  • ダウンロードサイズ: 6.7 MB

  • 必要ディスク容量: 72 MB

  • 概算ビルド時間: 0.9 SBU (using parallelism=4)

KWave の依存パッケージ

必須

AudioFile-0.3.6, fftw-3.3.10, id3lib-3.8.3, KDE Frameworks-5.113.0

推奨

alsa-lib-1.2.10, FLAC-1.4.3, libogg-1.3.5, PulseAudio-16.1

任意

Doxygen-1.9.8

KWave のインストール

以下のコマンドを実行して KWave をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: kwave
インストールライブラリ: libkwave.so, libkwavegui.so, 29 個のプラグイン
インストールディレクトリ: $KF5_PREFIX/lib/plugins/kwave, $KF5_PREFIX/share/kwave, $KF5_PREFIX/share/doc/HTML/*/kwave

概略説明

kwave

KDE フレームワーク 5 上にて構築されているサウンドエディター。
