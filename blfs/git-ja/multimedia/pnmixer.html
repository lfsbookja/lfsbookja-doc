pnmixer-0.7.2

Pnmixer の概要

Pnmixer パッケージはトレイアイコン内での軽量なボリューム制御を実現します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://github.com/nicklan/pnmixer/releases/download/v0.7.2/pnmixer-v0.7.2.tar.gz

  • ダウンロード MD5 sum: e9f17f56c50de39393030a96e343427b

  • ダウンロードサイズ: 236 KB

  • 必要ディスク容量: 2.5 MB

  • 概算ビルド時間: 0.1 SBU 以下

Pnmixer の依存パッケージ

必須

alsa-utils-1.2.10, GTK+-3.24.38, libnotify-0.8.3

Pnmixer のインストール

以下のコマンドを実行して Pnmixer をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr ..  &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: pnmixer
インストールライブラリ: なし
インストールディレクトリ: /usr/share/pnmixer
