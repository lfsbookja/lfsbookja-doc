テキストウェブブラウザー

Unix 互換システムに不慣れな方からは、よく以下のような質問を受けます。 「テキストブラウザーって、ほんとに必要なんですか？ 私は X もビルドしたし、 Firefox/Falkon などもビルドして使っているんですよ。」 Those who have been around systems for a while know that when (not if) you manage to mess up your graphical browser install and you need to look up some information on the web, a console based browser will save you. Also, there are quite a few people who prefer to use one of these browsers as their primary method of browsing; either to avoid the clutter and bandwidth which accompanies images or because they may use a text-to-speech synthesizer which can read the page to them (of use for instance to partially sighted or blind users).
