libndp-1.8

libndp の概要

libndp パッケージは IPv6 近隣探索プロトコル (Neighbor Discovery Protocol) のラッパーを提供します。 また NDP メッセージの送受信を行う ndptool というツールも提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): http://libndp.org/files/libndp-1.8.tar.gz

  • ダウンロード MD5 sum: c7e775fd5a9d676e8cba9c3732c4df93

  • ダウンロードサイズ: 360 KB

  • 必要ディスク容量: 2.5 MB

  • 概算ビルド時間: 0.1 SBU 以下

libndp のインストール

以下のコマンドを実行して libndp をビルドします。 

./configure --prefix=/usr        \
            --sysconfdir=/etc    \
            --localstatedir=/var \
            --disable-static     &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: ndptool
インストールライブラリ: libndp.so
インストールディレクトリ: なし

概略説明

ndptool

NDP メッセージの送受信を行うツールです。

libndp.so

IPv6 近隣探索プロトコル (Neighbor Discovery Protocol) のラッパーを提供します。
