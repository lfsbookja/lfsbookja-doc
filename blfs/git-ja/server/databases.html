データベース

本章ではデータベースについて説明します。 データベースには、シングルユーザーによる入出力の方式から、商用的なもので、データベースサーバーやトランザクションをサポートするものまでさまざまです。 通常は、他のアプリケーションに対する依存パッケージとして、データベースが必要となることから、本章に目を向けることになったことでしょう。 ただし SQL サーバーだけは、LFS システム上に構築することは不可能です。
