メールサーバーソフトウェア

MTA というのは、電子メールを一つのマシンから他のマシンへ転送するプログラムを指します。 古くからある MTA として Sendmail がありますが、他にもソフトウェアがいくつかあります。

SMTP サーバーに加えて、他にも POP/IMAP サーバー (Dovecot) があります。
