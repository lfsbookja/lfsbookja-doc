twm-1.0.12

twm の概要

twm パッケージは、非常に軽量なウィンドウマネージャーを提供します。

本パッケージは、Xorg のインストールが完成した際のテストのために提供されています。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://www.x.org/pub/individual/app/twm-1.0.12.tar.xz

  • ダウンロード MD5 sum: 805ee08b5a87e1103dfe2eb925b613b4

  • ダウンロードサイズ: 260 KB

  • 必要ディスク容量: 3.8 MB

  • 概算ビルド時間: 0.1 SBU 以下

twm の依存パッケージ

必須

Xorg-Server-21.1.10

twm のインストール

以下のコマンドを実行して twm をビルドします。 

sed -i -e '/^rcdir =/s,^\(rcdir = \).*,\1/etc/X11/app-defaults,' src/Makefile.in &&
./configure $XORG_CONFIG &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

コマンド説明

sed -i -e '/^rcdir =/s...: 本コマンドは twm の設定ファイルを適切なディレクトリにインストールするようにします。

パッケージ構成

インストールプログラム: twm
インストールライブラリ: なし
インストールディレクトリ: /etc/X11/app-defaults

概略説明

twm

X ウィンドウシステムにおけるタブウィンドウマネージャー (Tab Window Manager) です。
