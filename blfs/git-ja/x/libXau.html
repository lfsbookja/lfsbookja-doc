libXau-1.0.11

libXau の概要

libXau パッケージは X11 認証プロトコル (X11 Authorization Protocol) を実装したライブラリを提供します。 これは、クライアントにおいて表示アクセスを制限する際に利用されます。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://www.x.org/pub/individual/lib/libXau-1.0.11.tar.xz

  • ダウンロード MD5 sum: 7f14ba9c84a81a2b9dd023706febab38

  • ダウンロードサイズ: 268 KB

  • 必要ディスク容量: 2.8 MB (with test)

  • 概算ビルド時間: 0.1 SBU 以下 (テスト込み)

libXau の依存パッケージ

必須

xorgproto-2023.2

libXau のインストール

以下のコマンドを実行して libXau をビルドします。 

./configure $XORG_CONFIG &&
make

ビルド結果をテストする場合は make check を実行します。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libXau.so
インストールディレクトリ: なし

概略説明

libXau.so

X 認証データベース処理ライブラリです。
