adwaita-icon-theme-45.0

Introduction to Adwaita Icon Theme

The Adwaita Icon Theme package contains an icon theme for GTK+ 3 and GTK 4 applications.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

Package Information

  • Download (HTTP): https://download.gnome.org/sources/adwaita-icon-theme/45/adwaita-icon-theme-45.0.tar.xz

  • Download MD5 sum: 5407d7921f717e837341bca9bb1bc50b

  • Download size: 4.1 MB

  • Estimated disk space required: 72 MB

  • Estimated build time: 0.1 SBU

Adwaita Icon Theme Dependencies

Required

GTK+-3.24.38 or GTK-4.12.4, and librsvg-2.57.1

Optional

git-2.43.0, Inkscape-1.3.2, and Icon Tools

Installation of Adwaita Icon Theme

Install Adwaita Icon Theme by running the following commands: 

mkdir build &&
cd    build &&

meson setup --prefix=/usr .. &&
ninja

This package does not come with a test suite.

Now, as the root user: 

ninja install

Contents

Installed Programs: None
Installed Library: None
Installed Directory: /usr/share/icons/Adwaita
