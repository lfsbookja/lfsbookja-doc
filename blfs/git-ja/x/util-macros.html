util-macros-1.20.0

util-macros の概要

util-macros パッケージは Xorg のすべてのパッケージにて用いられる m4 マクロを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://www.x.org/pub/individual/util/util-macros-1.20.0.tar.xz

  • ダウンロード MD5 sum: 50135407d81e2c97c2879a2ba3bac688

  • ダウンロードサイズ: 84 KB

  • 必要ディスク容量: 520 KB

  • 概算ビルド時間: 0.1 SBU 以下

util-macros の依存パッケージ

必須

Xorg build environment (本節での内容が適切に動作するように設定が必要)

util-macros のインストール

以下のコマンドを実行して util-macros をビルドします。 

./configure $XORG_CONFIG

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: なし
インストールディレクトリ: $XORG_PREFIX/share/pkgconfig, $XORG_PREFIX/share/util-macros
