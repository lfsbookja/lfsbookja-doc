xcb-util-image-0.4.1

xcb-util-image の概要

xcb-util-image パッケージは XCB ライブラリに対して追加の拡張ライブラリを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://xcb.freedesktop.org/dist/xcb-util-image-0.4.1.tar.xz

  • ダウンロード MD5 sum: a67bfac2eff696170259ef1f5ce1b611

  • ダウンロードサイズ: 284 KB

  • 必要ディスク容量: 2.9 MB

  • 概算ビルド時間: 0.1 SBU 以下

xcb-util-image の依存パッケージ

必須

xcb-util-0.4.1

任意

Doxygen-1.9.8 (ドキュメントのため)

xcb-util-image のインストール

以下のコマンドを実行して xcb-util-image をビルドします。 

./configure $XORG_CONFIG &&
make

ビルド結果をテストする場合は LD_LIBRARY_PATH=$XORG_PREFIX/lib make check を実行します。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libxcb-image.so
インストールディレクトリ: なし

概略説明

libxcb-image.so

Xlib の XImage、xShmImage 関数のポートです。
