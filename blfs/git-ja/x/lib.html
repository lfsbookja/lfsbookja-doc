グラフィック環境ライブラリ

本章では X を動作させるために必要となるライブラリは説明しません。 ここで説明するのは X の機能を拡張するライブラリです。 In some cases the enhancement is as simple as font support. In others it is as complex as libraries that sit between X and applications that run on X whose purpose is to standardize the look and feel and inter-process communications for different applications. They also assist programmers by supplying common elements.
