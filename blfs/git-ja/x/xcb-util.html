xcb-util-0.4.1

xcb-util の概要

xcb-util パッケージは XCB ライブラリに対する拡張機能を提供します。 たいていの機能は Xlib に存在するものですが、ただし X プロトコルのコアな部分には含まれないものです。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://xcb.freedesktop.org/dist/xcb-util-0.4.1.tar.xz

  • ダウンロード MD5 sum: 34d749eab0fd0ffd519ac64798d79847

  • ダウンロードサイズ: 261 KB

  • 必要ディスク容量: 2.6 MB

  • 概算ビルド時間: 0.1 SBU 以下

xcb-util の依存パッケージ

必須

libxcb-1.16

任意

Doxygen-1.9.8 (ドキュメントのため)

xcb-util のインストール

以下のコマンドを実行して xcb-util をビルドします。 

./configure $XORG_CONFIG &&
make

このパッケージにテストスイートはありません。

root になって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libxcb-util.so
インストールディレクトリ: なし

概略説明

libxcb-util.so

他の XCB ユーティリティーに対して、ユーティリティー関数を提供します。
