xcb-util-cursor-0.1.4

xcb-util-cursor の概要

xcb-util-cursor パッケージは、XCB cursor ライブラリを実装したモジュールを提供します。 これは XCB による libXcursor の代替ソフトウェアです。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://xcb.freedesktop.org/dist/xcb-util-cursor-0.1.4.tar.xz

  • ダウンロード MD5 sum: 0d244518ad54b886413fe782235d6210

  • ダウンロードサイズ: 260 KB

  • 必要ディスク容量: 2.5 MB

  • 概算ビルド時間: 0.1 SBU 以下

xcb-util-cursor の依存パッケージ

必須

xcb-util-image-0.4.1 and xcb-util-renderutil-0.3.10

任意

Doxygen-1.9.8 (ドキュメントのため)

xcb-util-cursor のインストール

以下のコマンドを実行して xcb-util-cursor をビルドします。 

./configure $XORG_CONFIG &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libxcb-cursor.so
インストールディレクトリ: なし

概略説明

libxcb-cursor.so

Xlib の libXcursor 関数をポートしたものです。
