yelp-xsl-42.1

Yelp XSL の概要

The Yelp XSL package contains XSL stylesheets that are used by the Yelp help browser to format Docbook and Mallard documents.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.gnome.org/sources/yelp-xsl/42/yelp-xsl-42.1.tar.xz

  • ダウンロード MD5 sum: c41858c78e34bb8b68a535657a3e15d9

  • ダウンロードサイズ: 652 KB

  • 必要ディスク容量: 10 MB

  • 概算ビルド時間: 0.1 SBU 以下

Yelp XSL の依存パッケージ

必須

libxslt-1.1.39, itstool-2.0.7

Yelp XSL のインストール

以下のコマンドを実行して Yelp XSL をビルドします。 

./configure --prefix=/usr

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: なし
インストールディレクトリ: /usr/share/yelp-xsl
