gnome-calculator-45.0.2

GNOME Calculator の概要

GNOME Calculator はグラフィカルな計算機であり、財務計算、論理計算、科学計算など各種のモードを有する強力なものです。 倍精度演算パッケージを活用して、高精度な演算結果を実現します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

  • ダウンロード MD5 sum: cd6a59eb2b34be4c75dab07258cc6e47

GNOME Calculator の依存パッケージ

必須

gtksourceview5-5.10.0, itstool-2.0.7, libadwaita-1.4.2, libgee-0.20.6, libsoup-3.4.4

推奨

Vala-0.56.14

GNOME Calculator のインストール

以下のコマンドを実行して GNOME Calculator をビルドします。 

mkdir build &&
cd    build &&

meson setup --prefix=/usr --buildtype=release .. &&
ninja

ビルド結果をテストする場合は ninja test を実行します。

root ユーザーになって以下を実行します。 

ninja install

パッケージ構成

インストールプログラム: gcalccmd and gnome-calculator
インストールライブラリ: libgcalc-2.so and libgci-1.so
インストールディレクトリ: /usr/include/gcalc-2, /usr/include/gci-2, /usr/share/devhelp/books/{GCalc-2,GCi-1}, /usr/share/help/*/gnome-calculator

概略説明

gnome-calculator

GNOME デスクトップにおける公式の計算機です。

gcalccmd

gnome-calculator のコマンドライン版です。
