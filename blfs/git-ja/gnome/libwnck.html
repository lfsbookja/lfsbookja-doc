libwnck-43.0

libwnck の概要

libwnck パッケージは、ウィンドウナビゲーター構築キット (Window Navigator Construction Kit) を提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.gnome.org/sources/libwnck/43/libwnck-43.0.tar.xz

  • ダウンロード MD5 sum: cd21ef743a1e9286554401c5b28d5ec6

  • ダウンロードサイズ: 448 KB

  • 必要ディスク容量: 13 MB

  • 概算ビルド時間: less than 0.1 SBU

libwnck の依存パッケージ

必須

GTK+-3.24.38

推奨

gobject-introspection-1.78.1, startup-notification-0.12

任意

GTK-Doc-1.33.2

libwnck のインストール

以下のコマンドを実行して libwnck をビルドします。 

mkdir build &&
cd    build &&

meson setup --prefix=/usr --buildtype=release .. &&
ninja

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

ninja install

コマンド説明

--buildtype=release: 本パッケージの安定版リリース向けに適したビルドタイプを指定します。 デフォルトにしていると、最適化されていないバイナリを生成する場合があるためです。

-Dgtk_doc=true: Use this option to build the API reference manual.

パッケージ構成

インストールプログラム: wnckprop, wnck-urgency-monitor
インストールライブラリ: libwnck-3.so
インストールディレクトリ: /usr/include/libwnck-3.0, /usr/share/gtk-doc/html/libwnck-3.0

概略説明

wnckprop

スクリーン/ワークスペース/ウィンドウのプロパティを表示したり更新したりします。

libwnck-3.so

ページャー (pagers) やタスクリストへの更新を行う関数を提供します。
