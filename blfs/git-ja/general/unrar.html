UnRar-6.2.12

UnRar の概要

UnRar パッケージは RAR 形式ファイルの抽出ユーティリティを提供します。 RAR アーカイブからファイルを抽出する際に利用します。 RAR アーカイブは、主に Windows 環境において WinRAR というアプリケーションを使って生成することができます。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://www.rarlab.com/rar/unrarsrc-6.2.12.tar.gz

  • ダウンロード MD5 sum: 1f526429e8e50a9b3b94f325b2848c24

  • ダウンロードサイズ: 244 KB

  • 必要ディスク容量: 2.3 MB

  • 概算ビルド時間: 0.1 SBU

UnRar のインストール

注記

This package extracts the tarball to the unversioned directory unrar and not the expected directory unrar-6.2.12.

以下のコマンドを実行して UnRar をビルドします。 

make -f makefile

このパッケージにテストスイートはありません。

root になって以下を実行します。 

install -v -m755 unrar /usr/bin

パッケージ構成

インストールプログラム: unrar
インストールライブラリ: なし
インストールディレクトリ: なし

概略説明

unrar

RAR 形式アーカイブを解凍します。
