libyaml-0.2.5

libyaml の概要

yaml パッケージは YAML (YAML Ain't Markup Language) コードの解析と出力を行う C ライブラリを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

  • ダウンロード MD5 sum: bb15429d8fb787e7d3f1c83ae129a999

  • 必要ディスク容量: 6.4 MB （テスト込み）

  • 概算ビルド時間: 0.1 SBU 以下 （テスト込み）

libyaml のインストール

以下のコマンドを実行して libyaml をビルドします。 

./configure --prefix=/usr --disable-static &&
make

ビルド結果をテストする場合は make check を実行します。

root ユーザーになって以下を実行します。 

make install

コマンド説明

--disable-static: このスイッチはスタティックライブラリをインストールしないようにします。

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libyaml.so
インストールディレクトリ: なし

概略説明

libyaml.so

YAML コードを解析して出力する API 関数を提供します。
