sysmon-qt-1.0

sysmon-qt の概要

The sysmon-qt package provides a program to monitor system values including CPU usage, memory usage, and system temperatures in a compact screen window.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://github.com/lfs-book/sysmon-qt/releases/download/v1.0/sysmon-qt-1.0.tar.xz

  • ダウンロード MD5 sum: 724d50419b75f0941a0f4f22930b7440

  • ダウンロードサイズ: 76 KB

  • 必要ディスク容量: 1.3 MB

  • 概算ビルド時間: 0.2 SBU

sysmon-qt の依存パッケージ

必須

(Qt-5.15.11 または qt-alternate-5.15.11)

推奨

lm-sensors-3-6-0 (実行時)

sysmon-qt のインストール

以下のコマンドを実行して sysmon-qt をビルドします。 

cd src              &&
qmake sysmon-qt.pro &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

cp sysmon-qt                 /usr/bin                 &&
cp ../data/sysmon-qt.desktop /usr/share/applications/ &&
cp ../data/sysmon-qt.png     /usr/share/pixmaps/

パッケージ構成

インストールプログラム: sysmon-qt
インストールライブラリ: なし
インストールディレクトリ: なし
