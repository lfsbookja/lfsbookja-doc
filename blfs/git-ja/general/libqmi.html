libqmi-1.30.8

libqmi の概要

The libqmi package contains a GLib-based library for talking to WWAN modems and devices which speak the Qualcomm MSM Interface (QMI) protocol.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://www.freedesktop.org/software/libqmi/libqmi-1.30.8.tar.xz

  • ダウンロード MD5 sum: 5ec8838914f80e1dfa4d2fa8cc2f186d

  • ダウンロードサイズ: 1.2 MB

  • 必要ディスク容量: 151 MB (テスト込み)

  • 概算ビルド時間: 0.3 SBU (parallelism=4 利用時; テスト込み)

libqmi の依存パッケージ

必須

GLib-2.78.3, libgudev-238

推奨

gobject-introspection-1.78.1, libmbim-1.26.4

任意

GTK-Doc-1.33.2, help2man, libqrtr-glib

libqmi のインストール

以下のコマンドを実行して libqmi をビルドします。 

PYTHON=python3 ./configure --prefix=/usr --disable-static &&
make

ビルド結果をテストする場合は make check を実行します。

root ユーザーになって以下を実行します。 

make install

コマンド説明

--disable-static: このスイッチはスタティックライブラリをインストールしないようにします。

--enable-gtk-doc: GTK-Doc をインストール済で API ドキュメントを再構築する場合に指定します。

--disable-mbim-qmux: This switch disables support for using a MBIM control device for QMI messages. Use this switch if you did not install libmbim.

パッケージ構成

インストールプログラム: qmicli, qmi-firmware-update, qmi-network
インストールライブラリ: libqmi-glib.so
インストールディレクトリ: /usr/include/libqmi-glib, /usr/share/gtk-doc/html/libqmi-glib

概略説明

qmicli

is an utility used to control QMI devices

qmi-firmware-update

is an utility used to perform firmware updates on QMI devices

qmi-network

is an utility used for simple network management of QMI devices

libqmi-glib.so

contains API functions for talking to WWAN modems and devices which speak the Qualcomm MSM Interface (QMI) protocol
