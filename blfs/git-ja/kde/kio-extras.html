kio-extras-23.08.4

kio-extras の概要

The kio-extras package contains additional components to increase the functionality of the KDE resource and network access abstractions.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/release-service/23.08.4/src/kio-extras-23.08.4.tar.xz

  • ダウンロード MD5 sum: 1dbe7bd028d44c07d0e7eb82f01d57dd

  • ダウンロードサイズ: 1.4 MB

  • 必要ディスク容量: 44 MB

  • 概算ビルド時間: 0.5 SBU (using parallelism=4)

kio-extras の依存パッケージ

必須

kdsoap-2.1.1, KDE Frameworks-5.113.0

任意

libtirpc-1.3.4, Samba-4.19.3, taglib-1.13.1, libmtp, libssh, OpenEXR

kio-extras のインストール

以下のコマンドを実行して kio-extras をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: なし
インストールライブラリ: libkioarchive.so と 32 個のプラグイン
インストールディレクトリ: なし
