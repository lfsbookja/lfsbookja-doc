kuserfeedback-1.3.0

Kuserfeedback の概要

The kuserfeedback package contains a framework for collecting user feedback for applications via telemetry and surveys.

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/kuserfeedback/kuserfeedback-1.3.0.tar.xz

  • ダウンロード MD5 sum: d5e787de6cf0001f23133694dbffd96f

  • ダウンロードサイズ: 228 KB

  • 必要ディスク容量: 24 MB

  • 概算ビルド時間: 0.2 SBU (using parallelism=4)

Kuserfeedback の依存パッケージ

必須

extra-cmake-modules-5.113.0, Qt-5.15.11

Kuserfeedback のインストール

以下のコマンドを実行して kuserfeedback をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      -DBUILD_TESTING=OFF         \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: UserFeedbackConsole, userfeedbackctl
インストールライブラリ: libKUserFeedbackCore.so, libKUserFeedbackWidgets.so
インストールディレクトリ: /usr/include/KUserFeedback, /usr/lib/qml/org/kde/userfeedback, /usr/lib/cmake/KUserFeedback, /usr/share/KDE/UserFeedbackConsole, /usr/share/qlogging-categories5
