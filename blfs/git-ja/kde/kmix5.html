KMix-23.08.4

KMix の概要

KMix パッケージは KDE5 におけるオーディオミキサーアプリケーションを提供します。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/release-service/23.08.4/src/kmix-23.08.4.tar.xz

  • ダウンロード MD5 sum: 70dbcb9504b0c5b03552f9a654ece23a

  • ダウンロードサイズ: 1.2 MB

  • 必要ディスク容量: 18 MB

  • 概算ビルド時間: 0.2 SBU (using parallelism=4)

KMix の依存パッケージ

必須

KDE Frameworks-5.113.0

推奨

alsa-lib-1.2.10

任意

libcanberra-0.30, PulseAudio-16.1

KMix のインストール

以下のコマンドを実行して KMix をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: kmix, kmixctrl, kmixremote
インストールライブラリ: libkmixcore.so
インストールディレクトリ: $KF5_PREFIX/share/kmix, $KF5_PREFIX/share/kxmlgui5/kmix, $KF5_PREFIX/share/doc/HTML/*/kmix

概略説明

kmix

is a mini audio mixer for kde

kmixctrl

is used to save or restore the settings of kmix

kmixremote

is a utility to mute, get and set volume levels
