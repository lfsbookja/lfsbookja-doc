Gwenview-23.08.4

Gwenview の概要

Gwenview は KDE における高速で簡単なイメージビューアーです。

注記

LFS や依存パッケージが本ブックに示す最新安定バージョンでなかった場合には、BLFS 開発版においては、パッケージのビルドや処理実行が適切に行われないことがあります。

パッケージ情報

  • ダウンロード (HTTP): https://download.kde.org/stable/release-service/23.08.4/src/gwenview-23.08.4.tar.xz

  • ダウンロード MD5 sum: 176a628e4aba077ef0e0a132ffac28df

  • ダウンロードサイズ: 6.6 MB

  • 必要ディスク容量: 51 MB

  • 概算ビルド時間: 0.6 SBU (using parallelism=4)

Gwenview の依存パッケージ

必須

Exiv2-0.28.1, kimageannotator-0.6.1, KDE Frameworks-5.113.0, Little CMS-2.14

推奨

libkdcraw-23.08.4

任意

KF5Kipi

Gwenview のインストール

以下のコマンドを実行して Gwenview をビルドします。 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

このパッケージにテストスイートはありません。

root ユーザーになって以下を実行します。 

make install

パッケージ構成

インストールプログラム: gwenview, gwenview_importer
インストールライブラリ: libgwenviewlib.so, gvpart.so
インストールディレクトリ: $KF5_PREFIX/share/{kxmlgui5/gvpart,gwenview,doc/HTML/*/gwenview}

概略説明

gwenview

KDE イメージビューアー。

gwenview_importer

is a Photo importer
